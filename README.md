# onlinecourse

This project is a Django-based Online Course web application created as part of a Coursera assignment.

## Features

* Course and lesson management
* Questions and choices for mock exams
* Exam submission and result calculation
* Django admin configuration
* Bootstrap-based course detail template

## Technologies Used

* Python
* Django
* Bootstrap
* HTML

## Project Structure

* models.py – Database models for courses, lessons, questions, choices, and submissions
* admin.py – Admin configuration
* views.py – Exam submission and result display
* urls.py – URL routing
* course_details_bootstrap.html – Template for displaying course details
